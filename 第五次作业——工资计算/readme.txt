main1.asm 和 head1.asm 分别定义了主程序逻辑和辅助功能的实现。以下是详细说明：

main1.asm 功能分析
1. 数据段与表段
STKSEG: 定义栈段，用于存储中间数据。
DATASEG:
包含年份字符串（YEAR_STRINGS）。
每年的总收入（TOTAL_REVENUE，双字类型）。
每年的员工数量（WORKERS_COUNT，字类型）。
TABLE: 预定义了一段表格数据的空间，后续被填充。
2. 主程序 MAIN
数据段与表段初始化:
设置 DS 和 ES 段寄存器指向数据段和表段。
数据插入逻辑:
年份插入（INSERT_YEARS）: 从 YEAR_STRINGS 中读取每个年份（4 字节）插入到表段，每 12 字节对齐。
收入插入（INSERT_REVENUE）: 从 TOTAL_REVENUE 读取双字收入插入表段，每 16 字节对齐。
员工数量插入（INSERT_WORKERS）: 从 WORKERS_COUNT 读取员工数量插入表段，每 16 字节对齐。
计算平均收入:
利用每年的总收入和员工数量计算平均收入。
结果存储在表段中。
表格输出:
年份输出（PRINT_YEAR）: 每年从 TABLE 中读取 4 字节年份并打印。
收入、员工数及平均收入输出:
调用 PRINT_NUM 打印数值，调用 PRINT_TAB 添加间距。
重复打印 21 次，覆盖所有年份数据。
程序退出:
使用 INT 21H 中断结束程序。

head1.asm 功能分析
1. 数字打印功能
PRINT_NUM:
将数字逐位转换为 ASCII 字符并输出。
若数字不足 10 位，会补充空格对齐。
DIVDW:
执行 32 位无符号除法操作。
DX:AX 是被除数，CX 是除数。
结果:
商存储在 AX，余数存储在 CX。
2. 辅助打印功能
PRINT_TAB:
打印 4 个空格，作为表格间距。
PRINT_NEWLINE:
打印换行符（13 和 10 对应回车和换行）。

代码用途与整体功能
数据存储与组织:
将多个年份的数据（年份、收入、员工数量）组织成表格。
计算与格式化:
通过员工人数计算人均收入，并将其格式化存储。
输出表格:
使用年份、收入、员工数和平均收入以表格形式逐行输出。
