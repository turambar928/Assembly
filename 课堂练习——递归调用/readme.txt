代码用于计算并输出用户输入的第 n 项斐波那契数。程序流程如下：
1. 显示提示信息：提示用户输入一个1到100之间的数字。
2. 读取输入：将用户输入的数字转换成整数并存储。
3. 计算斐波那契数：调用递归函数 FIBONACCI 计算该数字位置上的斐波那契值。
4. 输出结果：将计算结果输出到屏幕上。

代码逻辑概述
1. 数据段（DATAS SEGMENT）
定义了提示信息 Prompt 和换行符 NewLine，用于显示输入提示和换行。
2. 程序入口与段设置
设置数据段寄存器 DS，开始执行程序。
调用 INPUT 子程序读取用户输入，并将结果存入 BX，用于后续的斐波那契计算。
3. 输入子程序 (INPUT)
通过 DOS 中断 INT 21H 显示提示信息。
使用循环读取用户输入的每个字符，将其从 ASCII 转换为整数并累加到 BX。
循环读取字符直到输入完成，并将多位数字转换成一个十进制整数存入 BX。
4. 斐波那契递归计算 (FIBONACCI)
递归调用的方式计算第 n 项的斐波那契数：
当 n 为 0 或 1 时，直接返回 0 或 1。
否则，递归调用 FIBONACCI 计算 F(n-1) 和 F(n-2)，然后将两者相加。
结果存储在 BX 中。
5. 输出子程序 (OUTPUT)
将 BX 中的斐波那契结果转换为字符并逐位输出：
使用循环和 DIV 操作将数字逐位取出，压入栈以倒序存储。
然后逐位弹出栈并转换为 ASCII 字符输出。
